# MultiuserChat

### Task (dop)

Заменить собеседника на бота, который может дихотомией отгадывает загаданное число.

### Compile project:

```bash
javac -d ./out ./src/*.java
```

### Run project:

Start Server:

```bash
java -cp ./out Server 4000
```

Start ServerBot:

```bash
java -cp ./out ServerBot 4000
```

Start Client:

```bash
java -cp ./out Client localhost 4000
```
